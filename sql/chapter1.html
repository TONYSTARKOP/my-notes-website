What is SQL?

SQL (Structured Query Language) is a standard language for managing and manipulating relational databases. Whether youâ€™re querying data, updating records, or designing schemas, SQL is your go-to superhero!

Example:

Imagine a library where books are organized in rows and columns. SQL is like the librarian who fetches the exact book you need, updates its status, or even restructures the library!

Features of SQL

  • Data Querying: Extract information from databases.
  • Data Manipulation: Insert, update, or delete data.
  • Data Definition:> Define database schemas and structures.
  • Data Control: Manage user access and permissions.

SQL Syntax

SQL commands are broadly categorized into:

  • DQL (Data Query Language): SELECT
  • DML (Data Manipulation Language): INSERT, UPDATE, DELETE
  • DDL (Data Definition Language): CREATE, ALTER, DROP
  • DCL (Data Control Language): GRANT, REVOKE

Practical Example

CREATE TABLE Books (
    BookID INT PRIMARY KEY,
    Title VARCHAR(255),
    Author VARCHAR(255),
    YearPublished INT
);

This creates a table named Books with columns for BookID, Title, Author, and YearPublished.
