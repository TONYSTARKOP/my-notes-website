Introduction to Data Retrieval in SQL

Data retrieval is the most frequent operation in SQL, and the SELECT statement is its cornerstone. Whether youâ€™re retrieving a single row or millions of records, mastering these commands will make you a query-writing wizard.

1. SELECT Statement Basics

The SELECT statement retrieves data from a database. At its simplest, it fetches all columns and rows from a table. 

SELECT * FROM table_name;

What it does: Fetches all data from the specified table.

2. Filtering Data with WHERE

The WHERE clause filters rows that meet a condition, making your queries precise and relevant. 

SELECT column1, column2 
FROM table_name 
WHERE condition;

What it does: Selects rows where the condition is true.

3. Logical Operators: AND, OR, NOT

Logical operators combine multiple conditions in a WHERE clause. 

SELECT column1 
FROM table_name 
WHERE condition1 AND condition2;

What it does: Selects rows where both conditions are true.

4. Sorting Results with ORDER BY

The ORDER BY clause sorts rows based on one or more columns. 

SELECT column1, column2
FROM table_name
ORDER BY column1 ASC;

What it does: Orders the results in ascending order based on the specified column.

5. Limiting Results with LIMIT

The LIMIT clause restricts the number of rows returned. 

SELECT column1 
FROM table_name 
LIMIT 10;

What it does: Retrieves only the first 10 rows.

6. Advanced Filters

SQL provides additional operators like BETWEEN, LIKE, and IN for complex filtering. 

SELECT column1 
FROM table_name 
WHERE column1 BETWEEN 10 AND 20;

What it does: Filters rows where column1 is between 10 and 20.
